I'm a software engineer from Durham, North Carolina and I'm currently focused on developing graph-based security tools at JupiterOne. I have a passion for building simple products that users love.

Notable Projects

Here are a few notable projects that I've developed or contributed to:

  • Kingdom Island - Kingdom Island was a multiplayer, cross-platform, virtual world for kids. Kingdom Island was available on iOS, Android, and in the web browser. Over 100,000 kids enjoyed chatting, playing mini games, obtaining in-game items, and exploring the island.
  • Toonkins - Toonkins was a multiplayer virtual world for kids developed with Unity. Players could choose from multiple different virtual characters and explore a fun exciting world.
  • Starbase - An open source security tool that collects assets and relationships from services and systems including cloud infrastructure, SaaS applications, security controls, and more into an intuitive graph view backed by the Neo4j database. Starbase supports over 100 open source integrations.
  • Marko - Marko is an open source project from eBay that reimagines HTML as a language for building dynamic and reactive user interfaces. Almost any valid HTML is valid Marko, and Marko extends HTML for building modern applications more declaratively. Among these extensions are conditionals and lists, state, and components.
  • Lasso - Lasso.js is an eBay open source Node.js-style JavaScript module bundler that also provides first-level support for optimally delivering JavaScript, CSS, images and other assets to the browser.
